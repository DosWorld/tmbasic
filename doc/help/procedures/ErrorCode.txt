.procedure ErrorCode
.category Error Handling
.blurb Gets the last error code that was thrown.
.description
Gets the last error code that was thrown.
Use this function in a `catch` block to determine the type of error that was thrown.

See {Built-In Constants:ref_constants} for a list of error codes used by TMBASIC functions.
.overload
.return Number
The last error code that was thrown.
.example
.example-code
try
    throw 5, "hello"
catch
    print ErrorCode
    print ErrorMessage
end try
.example-output
5
hello
.
