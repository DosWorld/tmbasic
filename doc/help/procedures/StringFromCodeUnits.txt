.procedure StringFromCodeUnits
.category Strings
.blurb Creates a string from a list of 16-bit Unicode code units.
.description
Creates a string from a list of 16-bit Unicode code units.
To convert the string back into a list of code units, use the p[CodeUnits] function.

Correct usage of this function requires deep Unicode knowledge.
Consider using the p[Characters] and p[Concat] functions instead.
.overload
.parameter input: String
The input string.
.return String
A list of the 16-bit Unicode code units in i[input].
.example
.example-code
dim x = CodeUnits("ABC")
print x(0)
print x(1)
print x(2)
.example-output
65
66
67
